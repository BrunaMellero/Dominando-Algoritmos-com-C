# 🚀 Dominando Algoritmos com C

Este repositório foi criado para organizar e compartilhar os exercícios desenvolvidos durante a **primeira etapa do curso de extensão em Linguagem C** da minha faculdade. O objetivo principal é consolidar os fundamentos de lógica e algoritmos utilizando uma das linguagens mais poderosas da computação.

## 📌 Projetos e Exercícios

Aqui estão os sistemas desenvolvidos nesta fase inicial:

1.  **Apresentação Pessoal**: Manipulação básica de strings e saída de dados.
2.  **Churrasco de Domingo**: Cálculo de insumos e divisão proporcional de custos.
3.  **Média de Notas**: Algoritmo para cálculo de média aritmética.
4.  **Compra no Exterior**: Conversor de moedas (Real para Dólar).
5.  **Compra Online**: Sistema de checkout com aplicação de desconto percentual.
6.  **Cuidando da Saúde (IMC)**: Calculadora de Índice de Massa Corporal.
7.  **Você lê rápido?**: Estimativa de tempo de leitura baseada em segundos/página.
8.  **Açaiteria**: Sistema de pedidos múltiplos com cupom de desconto.
9.  **Piloto de Kart**: Análise de performance e previsão de tempo de corrida.
10. **Meta Pessoal**: Planejamento financeiro com reserva fixa de 30% do saldo livre.

## 🛠️ Tecnologias e Aprendizados

* **Linguagem C**: Foco em tipos de dados (`int`, `float`, `char`).
* **Interatividade**: Uso de prompts claros para o usuário antes dos `scanf`.
* **Bibliotecas**: `<stdio.h>`, `<string.h>` e `<locale.h>`.
* **Formatação**: Controle de casas decimais e tratamento de strings com espaços (`fgets`).

## 🚀 Como executar os códigos

1.  Certifique-se de ter um compilador de C instalado (como o GCC).
2.  Clone o repositório:
    ```bash
    git clone [https://github.com/BrunaMellero/Dominando-Algoritmos-com-C.git](https://github.com/BrunaMellero/Dominando-Algoritmos-com-C.git)
    ```
3.  Navegue até a pasta, compile e execute o arquivo desejado:
    ```bash
    gcc nome_do_arquivo.c -o programa
    ./programa
    ```
    
## 📝 Aprendizados

Durante o desenvolvimento desses exercícios, foram colocados em prática conceitos como:
* Uso de `fgets` para leitura de strings com espaços.
* Formatação de saída com `printf` (limitação de casas decimais).
* Lógica matemática e conversão de unidades (metros para km, segundos para horas, etc).
* Tratamento de buffer de teclado.
